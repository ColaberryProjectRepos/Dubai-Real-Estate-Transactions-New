  1. Step 51. Adding the Rent Price KPI and sub indicators 

       
    1. -Right click on the KPI_Group Table Header and add new measure. In the formula bar enter the Average_Rent_Price Measure into the formula bar and click the check to the left of the formula bar. 
    2. -Click on a card visualization from the visualizations pane to the right of the screen 
    3. -Select the Card and drag the Average_Rent_Price to the fields slot for the card visual.




                        Step 52. Configuring the Rent Price KPI 
                                 
    1. -Now that we have setup the KPI, it is now time to configure it.
    2. -Ensure the KPI is selected on the dashboard and go to the format section in the visualizations pane.
    3. -Set to the following Configurations.
    Visual: Callout Value 
    Font = DIN  
    Size = 33
    Color = Conditional formatting (click the (fx)) and set the following configuration
    Format style = Rules
    What field should we base this on? = Average_Rent_Price
    --------These are the 3 rules to set--------
    If value =is blank then #FFFFFF
    If value >=  Min and <0 then #000000
    If value >=  Min and <0 then #000000 (once done click ok and continue)
    Display Units = Auto
    Value Decimal Places = Auto
    Text Wrap =On
    Source Spacing =On 
    Category label is off
